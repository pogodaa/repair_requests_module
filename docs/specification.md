# Спецификация к Заданию 1

## Входные данные
Данные поступают из файла `inputDataRequests.xlsx` (предоставлен заказчиком):

| Поле               | Описание |
|--------------------|---------|
| `requestID`        | Уникальный номер заявки |
| `startDate`        | Дата регистрации (ГГГГ-ММ-ДД) |
| `climateTechType`  | Тип оборудования (например, «Кондиционер») |
| `climateTechModel` | Модель устройства |
| `problemDescryption`| Описание неисправности |
| `requestStatus`    | Статус: «Новая заявка», «В процессе ремонта», «Готова к выдаче» |
| `completionDate`   | Дата завершения (может быть пустой) |
| `clientID`         | ID заказчика (ссылка на `inputDataUsers.xlsx`) |

## Выходные данные
Программный модуль должен вычислить и отобразить:

1. **Количество выполненных заявок** — где `requestStatus = "Готова к выдаче"`.
2. **Среднее время выполнения заявки** — в днях между `startDate` и `completionDate`.
3. **Статистику по типам оборудования** — сколько заявок по каждому `climateTechType`.

Эти данные отображаются на HTML-странице `statistics.html`.
