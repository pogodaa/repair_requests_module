# Детализация функции: Расчёт среднего времени выполнения заявки  
(в соответствии с ГОСТ 24.301)

## 1. Наименование функции  
`calculate_avg_repair_time`

## 2. Назначение  
Рассчитать среднюю продолжительность ремонта (в днях) для всех заявок со статусом «Готова к выдаче».

## 3. Входные данные  
Список заявок, где каждая содержит:
- `startDate` (строка, формат "ГГГГ-ММ-ДД")
- `completionDate` (строка или null)
- `requestStatus` (строка)

## 4. Алгоритм выполнения
1. Инициализировать переменные:  
   - `total_days = 0`  
   - `count = 0`  
2. Для каждой заявки:  
   2.1. Если `requestStatus == "Готова к выдаче"` и `completionDate` не null:  
       2.1.1. Преобразовать `startDate` и `completionDate` в объекты даты.  
       2.1.2. Вычислить разницу в днях: `days = completionDate - startDate`.  
       2.1.3. Если `days >= 0`, добавить `days` к `total_days`, увеличить `count` на 1.  
3. Если `count > 0`:  
   - `avg = total_days / count`  
4. Иначе:  
   - `avg = 0.0`  
5. Вернуть `round(avg, 2)`

## 5. Выходные данные  
Число с плавающей точкой — среднее время выполнения заявки в днях.

## 6. Особые условия  
- Пропускать заявки с некорректными датами.  
- Игнорировать заявки со статусом, отличным от «Готова к выдаче».
